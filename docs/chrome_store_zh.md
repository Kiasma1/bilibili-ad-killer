## 描述

Bilibili Ad Killer 是一款 Chrome 扩展程序，通过 AI 驱动的字幕分析自动检测并跳过 Bilibili 视频中的广告片段。该扩展程序实时监控视频字幕，使用 Google Gemini AI 通过分析字幕文本来识别广告内容，并提供自动跳过按钮以跳过广告片段，让用户享受不间断的视频观看体验。

主要功能：
- **AI 智能检测**：使用 Google Gemini AI 通过分析视频字幕、标题和描述来智能识别广告内容
- **自动跳过**：检测到广告时显示跳过按钮，让用户快速跳过广告片段
- **智能缓存**：记住已分析视频的广告时间范围，避免冗余的 AI 处理
- **可配置**：用户可以配置 Google Gemini API 密钥，选择 AI 模型，启用/禁用自动跳过，并设置视频时长过滤偏好

此扩展程序通过消除手动跳过广告片段的需要来增强 Bilibili 观看体验，节省时间并提供更流畅的视频观看体验。

## 单一用途

该扩展程序具有单一、明确的目的：通过使用 AI 技术分析视频字幕来自动检测并跳过 Bilibili 视频中的广告片段。该扩展程序仅在 Bilibili 视频页面上运行，不会修改或与任何其他网站或内容交互。

## 权限说明

**主机权限：**
- `https://www.bilibili.com/*`：需要访问 Bilibili 视频页面，扩展程序在此监控字幕并注入跳过功能
- `https://api.bilibili.com/*`：需要拦截来自 Bilibili 播放器 API（`api.bilibili.com/x/player/wbi/v2`）的 API 请求以访问视频字幕数据，这对于基于 AI 的广告检测至关重要

**存储权限：**
- `storage`：需要保存用户配置设置，包括 Google Gemini API 密钥、AI 模型偏好、自动跳过设置以及视频的缓存广告时间范围。此数据仅存储在本地，永远不会传输到第三方服务器。

**网络可访问资源：**
- `inject.js`、`lib/toastify.min.js`、`lib/toastify.min.css`：需要将广告检测和跳过功能注入到 Bilibili 视频页面中。这些资源仅在 `https://www.bilibili.com/*` 域名上可访问，对于扩展程序的核心功能是必需的。

所有权限对于扩展程序检测和跳过 Bilibili 视频中广告的单一目的都是直接必需的。未请求任何不必要的权限。
